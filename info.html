A comprehensive network and web reconnaissance tool designed for security professionals and developers.

Overview

imRobust is a versatile tool aimed at providing detailed information about network targets and web applications. It supports various reconnaissance techniques to gather essential data for security assessments and penetration testing.

Network Reconnaissance

The tool performs essential network-related tasks including pinging targets, performing DNS lookups, and retrieving Whois information. It can trace the route to a target to analyze network paths, provide geographical data about IP addresses, and identify domains hosted on the same server. Additionally, imRobust offers basic network sniffing capabilities on specified network interfaces to capture and analyze traffic.

SSL and Certificate Analysis

imRobust can retrieve SSL certificate details from servers and scan for common SSL/TLS vulnerabilities, helping to identify potential security weaknesses in encrypted connections.

Web Reconnaissance

For web application analysis, imRobust offers URL scanning with extraction of HTTP headers, cookies, links, forms, and scripts. It checks HTTP status codes, fetches important files like robots.txt and sitemap.xml, fingerprints HTTP headers, and traces redirect chains. The tool can also locate admin panels, verify allowed HTTP methods, and detect content management systems (CMS) used by websites.

Brute-Forcing and Enumeration

imRobust supports brute-force attacks on passwords and usernames using user-provided wordlists. It enumerates subdomains and directories, helping testers discover hidden or unlinked resources on a target web server.

Service Detection and Output

The tool detects services running on open ports to provide insight into the target’s active network services. Results can be saved in multiple formats such as JSON or CSV for further analysis and reporting.
